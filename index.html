HTML

HyperText Markup Language

  • Standard markup language

Features of HTML

  • HTML is highly flexible and simple
  • Support on almost every browser
  • User friendly

Disadvantages of HTML

  • Static page
  • Lots of code
  • Security not good

Tag

Text placed between '<' and '>' is called as tag

Example: <html>

Types of tag

  • Pair (<> </>)

    • The tags that have both opening and closing tags are called as paired tags.

    Example:

    • <html>
          ...
      </html>
    • <body>
          ...
      </body>
  • Single (</> / </>)

Note: closing tags start with '/'

Non-paired tags

  • The tags that have only opening tag but no closing tag are called as non-paired tags
  • Also called as self closed tags or forcefully tags or empty tags or singular tags

    • Example:

    • Break tag (line break)
      • <br> or <br/>
    • Horizontal rule (horizontal line)
      • <hr> or <hr/>
    • Image tag
      • <img> or <img/>

Structure of HTML

<html>
     <head>
         <title> </title>
     </head>
     <body> </body>
</html>

Working with HTML

  • Open any text editor for example Notepad++, Sublime Text, etc
  • Save this file with any name and extension '.html' or '.htm' for example 'index.html'
  • Run above file in any web browser for example MS Internet Explorer, MS Edge, Google Chrome etc

Note: HTML is case insensitive means A and a are same

Paired tag

Tag contains following tags:

  • Document title
    • <title>
  • Link to external document
    • <link/>
  • Meta information
    • <meta/>
  • Used for styling
    • <style>
  • Used to provide script
    • <script>

<link/>

Non-paired tag

Attributes:

  • href
  • rel
  • type

Example: Displaying favicon

  • Graphical image associated with title of web page
  • Its size should be 16 X 16 px or 23 X 32 px
  • <link href="anim01.gif" type="image/ico" rel="ico">
  • anim01.gif -> relative path

Paths

  • Absolute path
    • complete path or full path
  • Relative path
    • <link href="C:\Users\Pictures\" type="image/ico" rel="icon">
